# Roadmap

## Breve termine

- Strumento locale di re-identificazione dei risultati LLM usando il mapping riservato.
- Messaggi di errore ancora piu chiari per dipendenze mancanti.
- Piu esempi sintetici per fatture, contratti e liste fornitori.
- Screenshot della GUI e breve GIF demo.

## Medio termine

- Modalita batch da riga di comando.
- Export opzionale Markdown per casi selezionati.
- Migliore gestione di fogli Excel con piu tabelle.
- Campi fornitore/cliente configurabili.
- Eseguibile Windows con PyInstaller.

## Lungo termine

- OCR per documenti scansionati e immagini.
- Conservazione del layout originale DOCX/XLSX dove possibile.
- Mapping cifrati opzionali.
- Selettore lingua nella GUI.
- Dataset di valutazione per documenti aziendali italiani.
