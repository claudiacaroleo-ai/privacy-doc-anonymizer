# Roadmap

## Short term

- Local re-identification tool for LLM outputs using the reserved mapping.
- Cleaner error messages for missing dependencies.
- More synthetic examples for invoices, contracts, and supplier lists.
- GUI screenshots and short demo GIF.

## Medium term

- Command-line batch mode.
- Optional export as Markdown for selected use cases.
- Better handling of Excel sheets with multiple tables.
- Configurable supplier/customer field names.
- Packaged Windows executable with PyInstaller.

## Longer term

- OCR pipeline for scanned documents and images.
- Preserve original DOCX/XLSX formatting where feasible.
- Optional encrypted mapping files.
- Language selector for the GUI.
- Domain-specific evaluation set for Italian business documents.
